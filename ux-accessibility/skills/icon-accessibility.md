# Icon Accessibility
**Purpose:** Apply icon accessibility as a deliberate UX decision — informed by user research, platform conventions, and design principles.
## Core Framework
Accessibility is a design quality, not a compliance checkbox. Products that work for people with disabilities work better for everyone — curb cuts, closed captions, and voice control were all designed for accessibility and adopted universally.
## Principle
Every design decision should be traceable to a user need or a business objective. If the rationale for a decision is "it's best practice" or "it looks good," the decision hasn't been justified — it's been assumed.
## When to use
When this specific UX challenge is the active constraint.
