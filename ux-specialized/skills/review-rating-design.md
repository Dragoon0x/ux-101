# Review Rating Design
**Purpose:** Apply review rating design as a deliberate UX decision — informed by user research, platform conventions, and design principles.
## Core Framework
UX principles are universal. UX application is contextual. The heuristics that make a consumer app delightful might make an enterprise tool confusing. The patterns that work in e-commerce fail in healthcare. Domain expertise is the bridge between principles and practice.
## Principle
Every design decision should be traceable to a user need or a business objective. If the rationale for a decision is "it's best practice" or "it looks good," the decision hasn't been justified — it's been assumed.
## When to use
When this specific UX challenge is the active constraint.
