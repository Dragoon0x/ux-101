# UI Photography Direction
**Purpose:** Direct photography for use in interfaces — style, treatment, and the integration standards that make photos feel native to the UI.
## Core Framework
Visual design is not about making things pretty. It's about making things clear. The visual layer communicates hierarchy, state, relationships, and brand — all simultaneously.
## Principle
Every design decision should be traceable to a user need or a business objective. If the rationale for a decision is "it's best practice" or "it looks good," the decision hasn't been justified — it's been assumed.
## When to use
When this specific UX challenge is the active constraint.
