# Card Sorting
**Purpose:** Use card sorting to understand how users categorize information — informing information architecture and navigation design.
## Core Framework
Research is the foundation. Every design decision that isn't grounded in user understanding is a guess — and guesses at scale are expensive.
## Principle
Every design decision should be traceable to a user need or a business objective. If the rationale for a decision is "it's best practice" or "it looks good," the decision hasn't been justified — it's been assumed.
## When to use
When this specific UX challenge is the active constraint.
