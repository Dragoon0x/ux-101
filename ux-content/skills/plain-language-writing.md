# Plain Language Writing
**Purpose:** Write at a reading level appropriate for the broadest audience — typically 6th-8th grade for consumer products. Use short sentences (15-20 words). Use common words ('use' not 'utilize,' 'start' not 'initiate'). One idea per sentence. Active voice ('We'll send you an email' not 'An email will be sent'). Test readability with Hemingway Editor — aim for Grade 6-8. Plain language is not dumbed-down language — it's clear language.

## When to use
When writing or reviewing this type of interface text. When the current copy is unclear, too long, or not helping users accomplish their goals.
