# Placeholder Text
**Purpose:** Placeholders show format examples, not descriptions. Email field placeholder: 'name@company.com' — not 'Enter your email address.' Placeholders disappear on focus, so never put essential information in them. Placeholder color should be lighter than input text but still readable (minimum #757575 on white for accessibility). Some fields need no placeholder at all — if the label is clear, the placeholder adds nothing.

## When to use
When writing or reviewing this type of interface text. When the current copy is unclear, too long, or not helping users accomplish their goals.
