# Helper Text Design
**Purpose:** Helper text appears below input fields to prevent errors before they happen. One sentence maximum. Anticipate the user's question: 'We'll send a confirmation to this address' below the email field. Prevent the error: 'Must be 8+ characters with at least one number' below password. Helper text should be visible at all times, not just on focus. Gray color (#6B7280), 12-13px size, 4px below the input.

## When to use
When writing or reviewing this type of interface text. When the current copy is unclear, too long, or not helping users accomplish their goals.
