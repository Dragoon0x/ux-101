# Label Writing
**Purpose:** Labels identify form fields. They should be concise (1-4 words), descriptive (not generic), and always visible (never rely on placeholder text as the only label). Use nouns not questions: 'Email address' not 'What is your email?' Place labels above the field for fastest completion (research shows 50ms faster scanning than left-aligned labels). Labels should match the data format expected: 'Phone number' not 'Contact.'

## When to use
When writing or reviewing this type of interface text. When the current copy is unclear, too long, or not helping users accomplish their goals.
