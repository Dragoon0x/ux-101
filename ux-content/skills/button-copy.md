# Button Copy
**Purpose:** Button labels should describe the outcome, not the action. 'Create account' beats 'Submit.' 'Send invitation' beats 'Done.' Front-load the verb. Use sentence case (not ALL CAPS — it reduces reading speed). 1-3 words. The button label is often the most-read text on the screen — it deserves more writing time than the paragraph above it.

## When to use
When writing or reviewing this type of interface text. When the current copy is unclear, too long, or not helping users accomplish their goals.
