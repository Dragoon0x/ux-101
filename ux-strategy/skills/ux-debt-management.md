# UX Debt Management
**Purpose:** Identify, track, and prioritize UX debt — the design compromises that accumulate and eventually degrade the experience.
## Core Framework
Strategy is the layer that connects individual design decisions into a coherent system. Without strategy, design is a collection of well-crafted screens that don't add up to a product.
## Principle
Every design decision should be traceable to a user need or a business objective. If the rationale for a decision is "it's best practice" or "it looks good," the decision hasn't been justified — it's been assumed.
## When to use
When this specific UX challenge is the active constraint.
